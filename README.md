# TechShop - E-commerce Front-End Semântico 
Acesso do site [clique aqui](https://jadilson-tech.github.io/meu-projeto-ecommerce/)

Este é um projeto de uma plataforma de e-commerce moderna, desenvolvida para fins acadêmicos. O objetivo principal foi aplicar os conceitos de **HTML5 Semântico**, estilização avançada com **Transições CSS** e manipulação dinâmica do DOM através de **JavaScript Puro (Vanilla JS)**, além de integrar recursos de mídia externa de forma responsiva.

---

## Funcionalidades

* **Arquitetura Semântica:** Uso rigoroso das tags HTML5 (`<header>`, `<main>`, `<section>`, `<article>`, `<figure>`, `<footer>`) para otimização de acessibilidade e SEO.
* **Interface Interativa (UI/UX):** Layout moderno e limpo, inspirado nas melhores práticas de usabilidade do mercado, com foco na redução da fadiga visual do usuário.
* **Transições e Efeitos CSS:** * Efeito de flutuação (`translateY`) e sombreamento suave ao passar o mouse sobre os cards de produtos.
    * Efeito de *zoom* suave (`scale`) nas imagens dos produtos.
* **Dinamismo com JavaScript:** Sistema de interceptação de cliques que atualiza o contador do carrinho de compras em tempo real, fornecendo feedback visual instantâneo nos botões.
* **Mídia Integrada:** Incorporação asíncrona de vídeo do YouTube através de um container 100% responsivo (proporção 16:9).
